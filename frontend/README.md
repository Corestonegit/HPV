# Frontend - React + Vite

## Установка

```bash
# Установить зависимости
npm install
# или
yarn install
# или
pnpm install
```

## Запуск

```bash
# Запуск сервера разработки
npm run dev
# или
yarn dev
# или
pnpm dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в директории `dist`
